v0.9 防鸽子预览版  （很可能是最后一版了
本项目旨在帮助大家快速地预览大量特效模板
方便寻找所需的模板

用法大概是点击 “点我打开网页.cmd” 回自动打开一个网页
然后你挑选喜欢的模板样式
右键“在新标签页打开图片”
从网址中查看模板名称
最后去“ASS-模板”文件夹下寻找需要的模板


有啥问题可以b站私信：https://space.bilibili.com/472242
技术问题建议直接 github 开 issue


============================================
《如果卡住了！》
关掉打开的cmd窗口
再打开一次 “点我打开网页.cmd”

如果还是不行？
如果有打开的cmd窗口就关掉
看看任务栏有没有橙色的 mongoose 图标，有就先右键 Quit 退出
然后手动点击  mongoose.exe 他会自动最小化到任务栏上
之后在浏览器打开 http://localhost:2333 就可以看到网页了

关于网页
--------

使用了开源的框架 demo 改编，继续使用 MIT 协议

代码参见 https://github.com/inkydragon/Aegisub-Karaoke-Templates/blob/main/Ass-Fx-Viewer/

纯静态页面无法实现瀑布流加载，所以使用了本地的轻量级 http 服务器
专业用户可自行替换

macOS 和 Linux 用户请自行下载对应的 http 服务器可执行文件
https://mongoose.ws/
